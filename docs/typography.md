# Typography settings

Using typgraphers quotes should set "double quotes" and 'single quotes' to curly quotes.

## Special characters

- Copyright: `(c)` → (c)
- Trademark: `(tm)` → (tm)
- Ellipsis: `...` → ...
- Tankestrek: `--` → --
- Strikethrough: `~some text~` → ~some text~
